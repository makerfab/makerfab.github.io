用LCD智能控制器调试打印机

如果打印机有LCD智能控制器的话，可以通过LCD智能控制器调试打印机，否则，请使用Repitier Host 软件 或 Cura 软件联机调试。

确认打印胶条已经安装。

确认打印头在X轴和Y轴起始位置（保证打印头在整个台面的右后方位置，靠近后面板和右面板的位置），如果不是，需要掉电后手动归位，Z轴因为有停止位感应开关可以在控制软件中控制归位。

打印机上电

连接打印机电源给打印机上电

LCD智能控制器调试步骤

利用智能LCD按钮打开"Prepare -> Move Axis " 手动控制菜单：

  1. 测试Z轴运动：在菜单中选择Z，然后通过旋转按钮, 检查Z轴平台运动，逆时针旋转 , 随着读数加大，平台应该离打印头越来越远, 顺时针旋转，随着读数减小 , 平台应该离打印头越来越近.

  2. 测试X轴运动：在菜单中选择X，然后通过旋转按钮, 检查热头X向运动。

  3. 测试Y轴运动：在菜单中选择Y，然后通过旋转按钮, 检查热头Y向运动。

  4. 测试Z轴归位：通过 "Prepare -> Home" 菜单可以让XYZ归位(X=0,Y=0,Z=0), 归位以后，检查热头离平台距离是否符合要求，否则，你可能需要通过调节Z触碰件或Z平台的4个支撑螺栓调节平台高度，以便归位后让打印头正好触碰打印平台(一张薄纸的距离)。

  5. 测试热头：点击“Prepare -> Preheat PLA”加热挤出头 , 检查LCD首页的温度显示，看看温度有没有上升。

  6. 测试挤出机：温度上升到180C度后 , 通过 "Prepare -> Purge 5mm" 菜单，测试打印胶条是不是可以从热头里挤出。

  7. 测试风扇：温度上升到目标温度后，风扇会运转，通过"Prepare -> Cooldown " 冷却挤出头，看看风扇是否全力运转。
