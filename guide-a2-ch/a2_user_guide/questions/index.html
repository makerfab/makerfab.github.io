为什么我的挤出机无法挤出胶条？

挤出机无法挤出胶条有很多原因，可以一一做如下检查：

  1. 热头温度是不是到了可以挤出的温度，如果热头温度不到，挤出机是不工作的。
  2. 热头温度是不是合适，可以提高一下热头的温度试一下，比如对于PLA，提高到210C试试。
  3. 胶条是不是顺利进入热头，可以卸掉送料管固定盖检查一下。手动挤出试一下（要小心，因为手动挤出也是要热头加热到挤出温度的），换胶条的时候，最好卸掉送料管固定盖检查确定一下胶条是否顺利进入热头。
  4. 胶条是不是过粗，如果胶条太粗，比如比标准1.75mm粗很多（市面上有1.95当1.75mm卖的。），有些胶条粗细不均，也会有时不时不能挤出现象。
  5. 如果手动挤出没问题，看看挤出机弹簧够不够紧，可以试着把挤出机弹簧收紧一下看看。弹簧太紧或者太松都有可能无法挤出，如果太紧的话也要放松一些。
  6. 挤出齿是否磨损，如果挤出齿有磨损可能需要更换带挤出齿的挤出轴。
  7. 挤出机是否正常工作，如果挤出机有传动齿轮，而传动齿轮间压的太紧，可能就电机无法带动挤出齿轮工作。

为什么我的挤出机漏胶严重？

漏胶严重可能使用了比标准细的胶条，市面上有1.50mm的非标准胶条，不小心购入可能就会遇到问题。

也有可能是热头加热块和PEEK散热件之间的连接不够紧密，需要加固一下。
